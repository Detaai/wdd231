Chamber Directory

Project Overview

This site is the Chamber of Commerce directory for Kingman, Arizona. It displays local businesses and organizations, including contact info and membership levels. Users can switch between a grid view of member cards or a simple list view. All data is dynamically loaded from a JSON file.

City/Region: Kingman, Arizona
Purpose: Directory of local businesses
Features: Grid/List toggle, responsive layout
